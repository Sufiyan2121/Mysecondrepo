## Motivational Thought App (MERN - minimal)

This workspace includes a minimal MERN-style app:

- `server/` — Express backend (optional MongoDB via `MONGODB_URI`, otherwise local `data.json`).
- `client/` — React frontend (Vite).

Quick run (two terminals):

1. Start backend

```bash
cd server
npm install
npm run start
```

2. Start frontend

```bash
cd client
npm install
npm run dev
```

Open the client at `http://localhost:5173` and the server runs on `http://localhost:5000`.
